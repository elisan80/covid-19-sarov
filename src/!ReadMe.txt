Использоание программы:

Для корректной работы программы треубется наличие нескольких файлов с исходными данными:
humans_work.dat    - ИД по агентам для рабочих дней
humans_holiday.dat - ИД по агентам для выходных дней
seird.cfg          - ИД для SEIR модели (не обязательный)

Указанные файлы должны располагаться в одной папке (директория проекта). По умолчанию используется папка
./config/ располагающаяся рядом с исполняемым файлом.

Изменить папку проекта можно при помощи ключа '-p'. 

Примеры использования (разные варианты задания):

covid19.exe -p my_directory1/
covid19.exe -p c:/project/my_directory2
covid19.exe -p ../home/covid_data/my_directory3

Выходные файлы так же будут записываться в директорию проекта.
